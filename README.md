# Movie Database Exploration | SQL

## Project Overview
A SQL project exploring a real-world movie database to understand patterns in movies, ratings, and genres. The goal is to demonstrate core SQL concepts and practical data manipulation using structured queries.

## Project Steps

Created and structured the movie database with tables and relationships.

Performed joins, filters, aggregations, subqueries, and grouping to extract insights.

Updated records and created views for easier access to commonly queried data.

Explored data manipulation techniques and SQL query optimization.

## Tools Used
SQL, MySQL (or preferred RDBMS), Sample Movie Dataset

## Insights

Identified trends in movie ratings, genres, and release years.

Aggregated data revealed the most popular genres and top-rated movies.

Subqueries and grouping helped analyze patterns in ratings and movie counts by category.

## Result
The project demonstrates practical SQL skills, including database creation, querying, aggregation, and reporting, providing hands-on experience with real-world movie data.
